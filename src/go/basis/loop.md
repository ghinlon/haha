# Loop

```
for i := 0; i < l; i++ {
	do sth. 
}
```

* 从0开始数的, 循环体执行的次数是l次,若从1开始,循环体执行的次数就是l-1次  
  怎么理解这种东西呢, 不管从几开始数, i == l的那一次是不执行循环体的  
  所以循环体执行的次数就是从0或1到l-1次
* 不管从几开始数,i的最終状态等于l
