# Install

```sh
brew install cmus

--------------------------

pacman -S cmus alsa-lib 

~/.config/cmus/rc

set output_plugin=alsa
set dsp.alsa.device=default
set mixer.alsa.device=default
set mixer.alsa.channel=Master
```

# Doc

```sh
cmus		// launch

5		// file-browser
// under 5
a		// add files/folders

2		// library
// under 2
Enter		// play
c		// pause
<-/->		// seek by 10s
</>		// seek by 1m
z/b		// previous/next
y		// add to playlist

3		// Playlist

1		// Tree view
// under 1
<space>		// scroll to the artist
<tab>		// left/right column

7		// Settings

q		// quit

^R		// toggle repeat_current
D		// delete
```
