# 释义

原文:【廣韻】五合切【集韻】鄂合切，音。【玉篇】魚多貌。 　
又【集韻】魚口貌。 　
又【集韻】色洽切，音喢。【玉篇】以口歃飮。【淮南子·氾論訓】嘗一哈水，而甘苦知矣。【集韻】本作歃。或作。 　
又姓。【正字通】楊愼希姓有哈永森。 　
又【集韻】曷閤切，音合。與同。或作，齧聲也。 　
又呼合切，音。與欱同，大歠也。 　
又【集韻】【類篇】葛合切，音閤。與同，食也。 　
又託合切，音榻。義同。
